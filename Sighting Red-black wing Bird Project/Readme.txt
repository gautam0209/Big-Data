Readme

1. Project contains two scala source code files.
2. eBird.scala takes one parameter for input file and predicts the test error on 5% of same data split
3. eBirdPred.scala takes 3 parameters, 1 for trainingData File, 2 for testData File and 3 for output file. It will create the 
predictions on test data after getting trained on training Data.
4. Folder contains two syslog files, one for training and other for unlabeled data.
